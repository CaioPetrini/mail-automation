
# Mail Automation

Python desktop application for automated email sending using Excel spreadsheets and customizable HTML templates.

## Status

🚧 In development

## Goals

- Learn Python through a real-world project
- Apply software engineering best practices
- Build a desktop application with Tkinter
- Read and process Excel files
- Generate HTML emails
- Create a maintainable and scalable architecture

## Technologies

- Python
- Tkinter
- Pandas
- OpenPyXL
- JSON
